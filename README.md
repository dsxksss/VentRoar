## vite 脚手架搭配合食用

# 1、Quick start🚀🚀🚀

- 1-克隆此项目到本地 ✨

```
git clone https://gitee.com/xiao-kang-zhang/react-vite-creacte-my-demo.git
//克隆此项目文件
```

- 2-进入到克隆好的项目路径内,并安装本地依赖 ✨

```
cd react-vite-creacte-my-demo

npm install
//安装项目依赖库

```

- 3-启动 vite 自带开发服务器 ✨

```
npm run dev
//开启开发服务器
```

- 4 开启服务,进入 http://localhost:3000 🎉
